# Screenshots

This directory contains screenshots for the APK Purifier application.

## Required Screenshots

To complete the documentation, please add the following screenshots:

### 1. main-window.png
- Screenshot of the main APK Purifier window
- Should show the clean interface with "Add APK Files" button
- Include the menu bar and status bar
- Recommended size: 800x600 or higher

### 2. processing.png
- Screenshot showing the application during APK processing
- Should display the progress dialog with processing status
- Include log output showing the patching process
- Recommended size: 800x600 or higher

### 3. settings.png
- Screenshot of the Settings/Preferences dialog
- Should show the various configuration options:
  - Ad removal methods
  - Decompiler preferences
  - Output settings
- Recommended size: 600x500 or higher

## How to Take Screenshots

1. **Run the application**: `python src/main.py`
2. **Main Window**: Take a screenshot of the initial interface
3. **Settings**: Open Settings dialog and capture the configuration options
4. **Processing**: Add a test APK and start processing, capture during operation

## File Format
- Use PNG format for best quality
- Optimize file size (keep under 500KB each)
- Use descriptive filenames as specified above

## Notes
- Screenshots should be taken on a clean desktop background
- Ensure the application window is properly sized and centered
- Avoid including sensitive information in screenshots
